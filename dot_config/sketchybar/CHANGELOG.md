# SketchyBar 配置 - 版本历史

> Geek Glass 风格 - 半透明玻璃效果配置

## 版本说明

版本号格式：`主版本.次版本.修订版本`

- **主版本**: 重大架构变更
- **次版本**: 新功能或重要改进
- **修订版本**: Bug 修复和小改进

---

## [1.1.0] - 2026-03-19

### 优化与修复
- 清理 `icon_map.sh` 中的重复代码块（PDF 工具映射重复）
- 移除 `sketchybarrc` 中未使用的 `sep_cpu_mem_gap` 条目定义
- 优化 `input.sh` 调试日志配置（默认禁用，减少 I/O 开销）
- 统一所有插件文件的版本格式规范

### 文件变更
| 文件 | 变更类型 | 说明 |
|------|---------|------|
| `sketchybarrc` | 优化 | 清理注释代码，更新版本信息 |
| `plugins/icon_map.sh` | 修复 | 移除重复的 PDF 工具映射代码 |
| `plugins/input.sh` | 优化 | 日志默认禁用，条件调用减少开销 |
| `plugins/*.sh` | 文档 | 统一版本格式 |
| `CHANGELOG.md` | 新增 | 创建版本历史文件 |

## [1.0.0] - 2025-03-05

### 初始功能
- **AeroSpace 集成**: 工作区显示与切换（1-10）
- **系统监控**: CPU/MEM 进度条（彩虹配色）
- **电池显示**: 状态图标 + 百分比 + 进度条
- **时钟**: 时间（24h）+ 日期（英文月份）
- **天气**: Open-Meteo API（免费，无需 Key）
- **输入法**: Squirrel/Rime/ABC 检测
- **前台应用**: 图标 + 名称显示

### 视觉设计
- **Geek Glass** 风格：半透明玻璃 + 模糊效果
- **配色方案**:
  - 前景色: `0xFFF2F2F2` (接近白色)
  - 强调色: `0xFF5DC8E8` (亮青色)
  - 工作区聚焦: `0xFF7EDEFF` (天蓝色)
  - 栏背景: `0xAA3F3F3F` (67% 不透明度)

### 布局说明
```
[Logo] [1] [2] [3] ... [10] [前端应用图标 名称]  天气 • 内存 CPU 日期 时间 • 电池 • 输入法
```

## 颜色迁移历史

### v1.1.0 (2025-03-05)
- **ACCENT**: `0xFF1E7AA6` → `0xFF5DC8E8` (深青 → 亮青)
- **BAR_BG**: `0x703F3F3F` → `0xAA3F3F3F` (44% → 67% 不透明度)
  - 原因：提升深色背景下的对比度
